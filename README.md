O código acima é uma versão do term.ooo recriada utilizando o python 3.14, e algumas bilbiotecas, como pandas, colorama e unidecode, que devem ser instaladas para rodar o código.

Para instalar, abra seu terminal e digite os seguintes comandos:

python3.14 -m pip install --upgrade pip
python3.14 -m pip install pandas
python3.14 -m pip install colorama
python3.14 -m pip install unidecode
